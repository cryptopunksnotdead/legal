all with the same text formula:


```
Subject: [GitHub] DMCA takedown notice for cryptopunksnotdead/lets-go-programming-cryptopunks
Date: Feb 16th, 2023
From: support @ github.com
To:  gerald.bauer @ gmail.com

Hello,

We received a DMCA takedown notice regarding your repository
cryptopunksnotdead/lets-go-programming-cryptopunks.
As such, we have disabled public access to the repository.

The notice has been publicly posted at:

https://github.com/github/dmca/blob/master/2023/02/2023-02-14-morepunks.md

If you believe that your repository was disabled as a result of mistake
or misidentification, you have the right to file a counter notice and
have the repository reinstated. Our help articles provide more details
about how the DMCA notice-and-takedown process works at GitHub and how to
file a counter notice with us if you choose:

https://docs.github.com/articles/dmca-takedown-policy
https://docs.github.com/articles/guide-to-submitting-a-dmca-counter-notice

If you have any questions about the process or the risks in filing a
counter notice, we suggest that you consult with a lawyer.

Thanks,
The GitHub Team
```
